用法: portscanner -u <目标> -p <端口范围> -o <输出文件> -f <IP文件> -h
  -f string
        包含要扫描的IP地址的文件
  -h    显示帮助信息
  -o string
        保存扫描结果的文件
  -p string
        扫描的端口范围 (例如，'80,443' 或 '1-1024') (default "1-65535")
  -u string
        目标IP或URL

使用了golang的并发编程技术加持端口扫描，可以帮助渗透测试人员快速得到需要的信息，建立了多组工人池，针对于信息搜集和打点是个比较好用的辅助工具，比python多线程端口扫描的速度快
